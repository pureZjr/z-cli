# zjr-cli

zjr-cli 是一个适用简单并且可以快速构建 **_react、vue、小程序_** 的脚手架,适用于 macOS, Windows, 和 Linux 系统。

适用方式:

```sh
npm i -g zjr-cli
z-init 项目名称
cd 项目名称
npm i
```
